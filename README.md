### Kleuren
Pas de kleuren aan in de `:root` sectie bovenaan de `<style>` tag in `index.html`:

```css
:root {
    --primary: #1a1a1a;        /* Donkere tekst kleur */
    --secondary: #f5f5f5;      /* Lichte achtergrond */
    --accent: #ff6b6b;         /* Accent kleur (rood) */
    --text: #2d2d2d;           /* Tekst kleur */
    --text-light: #666;        /* Lichte tekst */
    --border: #e0e0e0;         /* Border kleur */
}
```
